# Project Summary
This project is focused on developing a premium LIGHT LOFT single-page website for Veniamin Levitin, a financial modeling and analysis expert. The website is designed to provide insights into investment mechanics, risk assessment, and various financial models, enabling users to navigate complex financial systems with clarity.

# Project Module Description
The website features several functional modules, including:
- **Hero Section**: Introduces Veniamin Levitin and his expertise.
- **Mechanics**: Explains how online investments work, detailing sources of income, payout structures, and risk factors.
- **Models**: Describes various investment models, including classic, high-yield, referral systems, MLM structures, and hybrid models.
- **For Whom**: Helps users identify if the content is suitable for them.
- **Results**: Showcases outcomes from applying the discussed financial models.
- **Methodology**: Outlines the analytical approach taken to assess financial models.
- **Red Flags**: Highlights common pitfalls and risks in investment strategies.
- **Mistakes**: Lists typical mistakes that can lead to financial loss.
- **Checklist**: Provides a pre-participation checklist for potential investors.
- **FAQ**: Answers common questions regarding investments and financial strategies.
- **Contacts**: Displays contact information for further inquiries.

- **Lucide React**: For icons used throughout the website.

# Usage
To set up the project locally, follow these steps:
1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies using `pnpm install`.
4. Build the project using `pnpm run build`.
5. Start the development server using `pnpm run dev`.
